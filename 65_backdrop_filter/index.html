CSS backdrop-filter

The backdrop-filter property applies graphical effects to everything behind an element, rather than the element itself. It's commonly used to create the modern "glassmorphism" effect.

Basic Syntax

.glass {
    background: rgba(255,255,255,.2);
    backdrop-filter: blur(10px);
}

Notice that the background is semi-transparent. If the element has a fully opaque background, you won't see the filtered backdrop.

Common Filter Functions

Function Description Example
blur() Blurs the background. blur(10px)
brightness() Makes the backdrop brighter or darker. brightness(150%)
contrast() Adjusts contrast. contrast(200%)
grayscale() Converts backdrop to grayscale. grayscale(100%)
invert() Inverts backdrop colors. invert(100%)
opacity() Adjusts backdrop opacity. opacity(50%)
saturate() Makes colors more or less vivid. saturate(200%)
sepia() Applies a sepia effect. sepia(100%)
hue-rotate() Rotates the color wheel. hue-rotate(90deg)

How It Works

Unlike the normal filter property, which affects the element itself, backdrop-filter affects whatever is visible behind the element. 

/* Filters the element */
filter: blur(10px);

/* Filters the background behind the element */
backdrop-filter: blur(10px);

Requirements

  • ✔ The element should have a transparent or semi-transparent background.
  • ✔ There must be visible content behind the element.
  • ✔ Usually paired with background: rgba(...).
  • ✔ Safari often requires -webkit-backdrop-filter.

Performance Notes

  • Backdrop filtering is more expensive than normal CSS backgrounds.
  • Avoid applying it to many large elements.
  • Blur is the most commonly used effect.
  • Keep blur values reasonable (typically 5–20px).
