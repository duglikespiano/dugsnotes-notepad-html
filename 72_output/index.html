HTML <output> Element

The <output> element represents the result of a calculation, user action, or script. It is commonly used together with forms, JavaScript, sliders, calculators, and interactive controls.

Basic Syntax

<output id="result">0</output>

Unlike a normal <span>, an <output> element has semantic meaning: it tells browsers and assistive technologies that this value is the result of some computation.

<output> Attributes

Attribute Description
id Allows JavaScript to locate the output.
name Name of the output within a form.
for Lists the IDs of elements that contributed to the result. Mostly used for semantics and accessibility.

<output> vs Similar Elements

Element Purpose Editable? Semantic Meaning
<output> Calculated result No Yes
<span> Generic inline container No No
<div> Generic block container No No
<input> User input Yes User-entered value
<meter> Measurement within a range No Represents a measured value
<progress> Task progress No Represents progress completion

The <output> element has excellent browser support and has been available across all major browsers for many years.

When Should You Use <output>?

  • ✔ Calculator results
  • ✔ Price totals
  • ✔ Live slider values
  • ✔ Form calculations
  • ✔ BMI or unit converters
  • ✔ Interactive educational tools
  • ✔ Any computed value generated by JavaScript

If the displayed value is the result of a calculation or user interaction, <output> is usually a better semantic choice than using a generic <span>.
