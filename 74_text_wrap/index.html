CSS text-wrap Property

The text-wrap property controls how text is wrapped onto multiple lines. Some values prioritize balanced line lengths, while others focus on preventing awkward breaks.

Browser Support
  • wrap and nowrap are widely supported.
  • balance is supported in modern Chrome, Edge, Safari, and newer Firefox versions.
  • pretty is newer and browser support is still improving.
  • stable is mainly intended for editable content and has limited support.
  • If your browser doesn't support a value, it will usually behave like wrap.

Comparison Table

Value Purpose Typical Use Browser Support
wrap Normal wrapping General text Excellent
nowrap No wrapping Buttons, labels, menus Excellent
balance Balanced line lengths Headings, article titles Good (modern browsers)
pretty More attractive line breaks Articles, paragraphs Still improving
stable Stable wrapping while editing Editors, contenteditable Limited

When should you use each?

  • wrap → Normal paragraphs (default).
  • nowrap → Navigation links, buttons, table headers.
  • balance → Headlines and page titles.
  • pretty → Long-form reading where typography matters.
  • stable → Rich text editors or contenteditable elements.
