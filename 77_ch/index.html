CSS ch Unit

The ch unit represents the width of the "0" (zero) character of the current font.

1. Different ch Widths

width:20ch
width:30ch
width:40ch
width:10ch;
width:20ch;
width:30ch;
width:40ch;

2. Compare ch vs px

20ch

The width depends on the font.

200px

The width is always 200 pixels.

4. Font Size Also Changes ch

16px font
24px font
36px font

Since ch is relative to the current font, larger font sizes create larger widths.

5. Best Practice: Readable Paragraph Width

Very long lines are difficult to read. Using max-width:65ch keeps each line around 65 characters, which is considered comfortable for reading.

6. Perfect for Input Fields

You can size an input based on approximately how many characters the user should type.





input{
    width:20ch;
}

7. Compare em, rem, %, vw and ch

Unit Meaning Based On
px Fixed pixels Screen pixels
em Relative Parent font size
rem Relative Root font size
% Relative Parent element
vw Relative Viewport width
ch Width of "0" Current font

8. ch Is NOT Character Count

iiiiiiiiiiiiiiiiiiii
WWWWWWWWWWWWWWWWWWWW

Both boxes are 20ch, but the text occupies different amounts of space because "i" is much narrower than "W". The width is based on the size of the 0 character, not on the average width of all characters.
