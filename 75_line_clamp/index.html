CSS line-clamp

The line-clamp property limits how many lines of text are displayed. Extra text is hidden after the specified number of lines.

Browser Support
  • Chrome ✅
  • Edge ✅
  • Safari ✅
  • Firefox ✅ (recent versions)

For the best compatibility, continue using -webkit-line-clamp, display:-webkit-box, and -webkit-box-orient:vertical.

Syntax

.clamped {
    overflow: hidden;

    display: -webkit-box;
    -webkit-box-orient: vertical;

    -webkit-line-clamp: 3;

    /* Optional modern syntax */
    line-clamp: 3;
}

Comparison Table

Property Effect Typical Use
No clamp Shows entire text Articles, documentation
line-clamp:1 Single line + ellipsis Titles
line-clamp:2 Maximum 2 lines Cards
line-clamp:3 Maximum 3 lines Blog previews
line-clamp:5 Maximum 5 lines Long descriptions

When Should You Use It?

  • ✅ Product cards
  • ✅ News websites
  • ✅ Blog previews
  • ✅ Social media feeds
  • ✅ Uniform grid layouts

Things to Remember

  • line-clamp only affects multi-line text.
  • Always combine it with overflow:hidden.
  • For maximum browser compatibility, include:
    • display:-webkit-box
    • -webkit-box-orient:vertical
    • -webkit-line-clamp
  • Unlike text-overflow: ellipsis, which works for a single line, line-clamp can truncate multiple lines.
  • The clipped text still exists in the DOM and can be copied or accessed by assistive technologies depending on the context.
