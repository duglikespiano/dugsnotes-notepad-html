CSS Relative Colors

What is a Relative Color?

A relative color lets you create a new color by starting from an existing color instead of writing every RGB or HSL value yourself.

Rather than saying:

background: hsl(220 70% 60%);

you can say:

background: hsl(from royalblue h s calc(l + 20));

This means:

  • Take the color royalblue.
  • Extract its HSL values.
  • Increase the lightness.
  • Create a new color.

General Syntax

hsl(
    from <color>
    h
    s
    l
)

rgb(
    from <color>
    r
    g
    b
)

lab(from ...)
lch(from ...)
oklab(from ...)
oklch(from ...)
color(from ...)

The from keyword tells CSS:

Take an existing color and expose its individual channels.

How It Works

Suppose our base color is:

:root {
    --brand: royalblue;
}

Then:

background:
    hsl(
        from var(--brand)
        h
        s
        calc(l + 20)
    );

means:

  1. Read --brand.
  2. Convert it into HSL.
  3. Keep Hue unchanged.
  4. Keep Saturation unchanged.
  5. Add 20 to Lightness.
  6. Create a new color.

Available Variables

Color Function Available Variables
rgb(from ...) r, g, b, alpha
hsl(from ...) h, s, l, alpha
lab(from ...) l, a, b, alpha
lch(from ...) l, c, h, alpha
oklab(from ...) l, a, b, alpha
oklch(from ...) l, c, h, alpha

Why Use Relative Colors?

Without Relative Colors With Relative Colors
Need to manually calculate new RGB/HSL values. Reuse an existing color.
Hard to maintain theme colors. Theme updates automatically.
Duplicate many color values. Only modify the channels you want.
Difficult to make lighter/darker variants. Use simple math with calc().
Easy to introduce inconsistent colors. Derived colors stay visually related.

Typical Use Cases

  • Create hover colors from the original button color.
  • Create disabled versions by lowering saturation.
  • Create dark mode colors automatically.
  • Generate borders slightly darker than backgrounds.
  • Create shadows with lower alpha.
  • Generate an entire color palette from one brand color.

Browser Support

Relative colors are part of the CSS Color Module Level 5 specification. Modern versions of Chrome, Edge, Safari, and Firefox have support, though some features may require up-to-date browser versions. If you need to support older browsers, consider providing a fallback color before the relative color declaration: 

.button {
    /* Fallback */
    background: royalblue;

    /* Modern browsers */
    background:
        hsl(from royalblue h s calc(l + 15));
}
